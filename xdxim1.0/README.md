### 一、运行demo

下载代码后，先配置好数据库（other包下面有数据库）。启动java项目后访问 **http:127.0.0.1:1997** 即可

<br/>

### 二、自己修改

如果你要修改你就要自己启动前端项目了，先安装node，然后安装npm。最后  npm run serve 即可。

<br/> 

### 三、其它

如果你修改完成之后也只想启动一个项目，就打包vue（npm run build）丢到java的static目录下面即可。（注意ip地址的修改）
